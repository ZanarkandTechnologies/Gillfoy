# Data Viz Gotchas

- **Responsiveness**: SVGs don't resize like divs. Use `viewBox` and `preserveAspectRatio` for responsive charts.
- **Color Blindness**: Use accessible color palettes (e.g., ColorBrewer) to ensure visualizations are inclusive.
- **Overplotting**: Too many data points on a chart lead to "ink-blobs." Consider downsampling or using density plots.
