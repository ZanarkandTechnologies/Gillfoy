# How to Plan Data Visualization Features

## Before You Start
1. Identify data shape and update frequency.
2. Choose chart type based on data relationship.
3. Determine interactivity requirements (hover, click, zoom).

## Planning Order
1. **Data layer**: Fetch, transform, normalize.
2. **Chart selection**: Bar, line, pie, scatter, custom.
3. **Interactivity**: Tooltips, legends, drill-down.
4. **Responsiveness**: Mobile-friendly sizing.

## Key Questions
- Is data static or real-time?
- How many data points (performance)?
- Are there comparison or aggregation needs?

## Common Patterns
- Dashboard layout with multiple charts
- Time series with zoom and pan
- Interactive knowledge graph with React Flow
