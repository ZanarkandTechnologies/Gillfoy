# How to Plan Cinematic Landing Pages

## Before You Start
1. Define the narrative arc (hook → story → CTA).
2. Identify scroll-triggered sections.
3. Determine asset requirements (images, videos, 3D).

## Planning Order
1. **Narrative structure**: Hero → problem → solution → proof → CTA.
2. **Scroll mechanics**: Pin points, scrub ranges, parallax layers.
3. **Asset generation**: AI-generated or curated visuals.
4. **Performance budget**: Target FPS, bundle size limits.

## Key Questions
- Is this a single-page scroll or multi-section?
- Are there WebGL/3D elements?
- What's the mobile fallback strategy?

## Common Patterns
- Hero with video background → scroll to reveal text
- Pinned section with scrubbed animation
- Parallax layers with depth effect
